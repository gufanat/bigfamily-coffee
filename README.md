# bigfamily-coffee
Це дипломний проєкт, який являє собою сучасний адаптивний веб-сайт для кав’ярні «BigFamily». Продукт розроблено для підвищення впізнаваності закладу, надання клієнтам актуальної інформації про послуги, меню, технології приготування кави та можливості онлайн-бронювання столиків.

Керівник роботи — М.В. Кожем’якін.

# Завдання дипломної роботи
Розробити веб-сайт для кав’ярні «BigFamily», який забезпечує ознайомлення користувачів із закладом, надає інформацію про кавові напої та десерти, дозволяє переглядати фотогалерею та контакти.

# Основні можливості
1. Інформація про заклад — ознайомлення з діяльністю та концепцією кав’ярні.
2. Кавова карта — перегляд асортименту кавових напоїв.
3. Меню десертів — представлення продукції закладу.
4. Галерея — перегляд фотографій інтер’єру та атмосфери кав’ярні.
5. Контактна інформація — адреса, контакти та соціальні мережі.
6. Адаптивний дизайн — коректна робота на ПК, планшетах та смартфонах.

# Використані технології
1. Мова розмітки — HTML5
2. Мова стилів — CSS3
3. Мова програмування — Vanilla JavaScript (ES6)
4. Прототипування інтерфейсу — Figma
5. Середовище розробки — Visual Studio Code
6. Керування версіями — GitHub

# Архітектура проєкту
Проєкт реалізований за клієнтською архітектурою та складається з трьох основних файлів:
1. index.html — структура веб-сторінки;
2. style.css — оформлення та адаптивний дизайн;
3. script.js — інтерактивність та обробка подій користувача.

# Основний функціонал
1. Навігація між секціями сайту;
2. Плавне прокручування сторінки;
3. Адаптивне меню;
4. Фільтрація контенту;
5. Інтерактивна галерея;
6. Анімації появи елементів;
7. Блок контактної інформації та соціальних мереж.
